每次游戏版本更新后如不能使用，请去下载最新版本。

涂装没显示解决办法，首先查看安装路径是否正确。
比如你有没有把profile文件夹复制到profile里面，这样是错误的。
必须把插件的profile文件夹和游戏的profile文件夹合并，跳出覆盖提示选“是”！！
其次打开游戏根目录的paths.xml，查看是否和以下示例符合，不符合请复制黏贴进去。
注意：0.5.8.0是游戏版本号，请根据游戏最新版本号更新
<?xml version="1.0"?>
<root>
	<Paths>
		<Path>res_mods/0.5.8.0</Path>
		<Path>profile</Path>
		<Path key="9f176">res_packages</Path>
		<Path key="a1269">res</Path>
	</Paths>
</root>
